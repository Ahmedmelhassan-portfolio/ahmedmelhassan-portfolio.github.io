# Ahmed Mohammed Elhassan Ahmed — Portfolio

Personal portfolio website of a civil engineer (B.Sc. 2026, Karadeniz Technical University).

## Host it on GitHub Pages (free)

1. Create a GitHub account if you don't have one.
2. Create a **new public repository** named exactly:
   `YOUR_USERNAME.github.io`
   (replace YOUR_USERNAME with your GitHub username — this makes the site live at that address).
3. Upload `index.html` (and the `images/` folder once you add photos) to the repository.
4. Wait 1–2 minutes. Your site is live at `https://YOUR_USERNAME.github.io`.

Alternatively, use any repository name (e.g. `portfolio`) and enable
**Settings → Pages → Deploy from branch → main / root**. The site will be at
`https://YOUR_USERNAME.github.io/portfolio`.

## Before publishing — checklist

- [ ] Add your photo as `images/profile.jpg` and swap the placeholder in the title block.
- [ ] Add project images: `images/buildtech.jpg`, `images/sap2000.jpg`,
      `images/insaatbotu.jpg`, `images/research.jpg` and replace the placeholder divs
      (each placeholder has a `<!-- REPLACE ... -->` comment showing the exact tag to paste).
- [ ] Upload your CV as `cv.pdf` and update the "CV (PDF)" button link in the hero.
- [ ] Replace `YOUR_USERNAME` in the GitHub and LinkedIn links in the footer.
- [ ] Make sure both Google Drive certificate files are shared as "Anyone with the link can view".
